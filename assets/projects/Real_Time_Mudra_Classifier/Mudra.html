Real-Time Hand Mudra Classification ü§ò

During the pandemic, my teammate faced challenges in learning classical dance mudras. To overcome this, we developed a real-time hand mudra classification system to assist in the learning process.

üîç Project Overview

  • Developed a real-time hand mudra classification system to aid in learning classical dance mudras.The system was designed to assist individuals, especially during the pandemic, in learning dance forms at home. Used computer vision and machine learning techniques to recognize and classify hand mudras in real-time.

Project Workflow

  • Data Collection:
    • Created a custom dataset by capturing diverse hand mudra poses through photography sessions.
  • Data Preprocessing:
    • Ensured dataset uniformity by resizing images, normalizing pixel values, and applying augmentation techniques.
  • CNN Model Development:
    • Developed a Convolutional Neural Network (CNN) architecture for effective image classification.
  • Transfer Learning:
    • Utilized transfer learning with pre-trained models (e.g., VGG16, ResNet) to enhance model performance.
  • Training and Validation:
    • Split the dataset into training and validation sets to train the model and prevent overfitting.
  • Fine-Tuning and Optimization:
    • Optimized hyperparameters and architecture, implemented learning rate schedules, and applied dropout.
  • Evaluation Metrics:
    • Evaluated the model using accuracy, precision, recall, and other metrics on a separate test dataset.
  • Integration with Real-Time System:
    • Integrated the trained model into a real-time system for live hand mudra classification.
  • Continuous Monitoring:
    • Implemented real-time monitoring to track system performance, user interactions, and classification accuracy.
  • Documentation and Collaboration:
    • Documented the entire process for knowledge sharing within the team and collaborated on insights and feedback to enhance model performance.
